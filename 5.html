Materi tentang PHP

Pengenalan PHP

PHP (Hypertext Preprocessor) adalah bahasa pemrograman server-side yang sering digunakan untuk pengembangan web.

Instalasi PHP

Anda dapat mengunduh PHP dari situs resmi PHP. Untuk pengembangan lokal, Anda dapat menggunakan XAMPP atau WAMP.

Struktur Dasar PHP

<?php
echo "Hello, World!";
?>

Variabel dan Tipe Data

$nama = "hannaaffiii";  // String
$umur = 17;      // Integer
$tinggi = 1.75;  // Float
$mahasiswa = Pelajar; // Boolean

Fungsi

function greet($name) {
    return "Hello, $name!";
}

echo greet("Hannaaffiii");

Struktur Kontrol

if ($umur >= 17) {
    echo "Dewasa";
} else {
    echo "Anak-anak";
}

Looping

for ($i = 0; $i < 5; $i++) {
    echo $i;
}

while ($umur < 30) {
    echo $umur;
    $umur++;
}

Array dan Associative Array

$array = array("apel", "jeruk", "pisang");

$associativeArray = array(
    "nama" => "Hannaaffiii",
    "umur" => 17
);
echo $associativeArray["nama"];
